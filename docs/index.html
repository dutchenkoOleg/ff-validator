# Rules

# required

Check the field is valid if it matters

Invalid cases:

  1. undefined
  2. null
  3. false - falsy boolean
  4. NaN - not a number
  5. [] - empty array
  6. "" - empty string

Valid cases:

  • true - truthy boolean
  • {} - any instance of Object
  • 0 / "0" - zero has matter as value
  • any other value that will be true in a logical transformation, e.g. !!myValue

Usage

                
                    import { required } from 'ff-validator/dist/rules/required';
                    const myRequiredRule = required('My custom error message');

                    // invalid cases             // valid cases
                    myRequiredRule();            myRequiredRule(true);
                    myRequiredRule(null);        myRequiredRule({});
                    myRequiredRule(false);       myRequiredRule(0);
                    myRequiredRule(NaN);         myRequiredRule('0');
                    myRequiredRule([]);
                    myRequiredRule('');
